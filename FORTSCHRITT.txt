  Frontend : React + Vite, dist/ EINGECHECKT (nach Änderungen: npm run build!)

  KI       : google-genai SDK (NICHT das alte google-generativeai – deprecated!)
             Modell: gemini-3.1-flash-lite (stabil)
  Docs     : Service Account, Env GOOGLE_SERVICE_ACCOUNT_JSON auf Render
